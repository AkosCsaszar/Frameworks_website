
# Frameworks Website
In this project I make a website with different Javascript frameworks: react, angular, vue. I also use Single Spa microfrontend to connect these framworks to in one website.

## Run the project

### homepage
```
cd homepage
npm i
npm run build
npm start
```

### navbar
```
cd navbar
npm i
npm run build
npm start
```

### root-config
```
cd root-config
npm i
npm run build
npm start
```

### react
```
cd react
npm i
npm run build
npm start
```

### angular

```
cd angular
npm i
npm run build
npm start
```

### vue
```
cd vue
npm i
npm run build
npm run serve
```
